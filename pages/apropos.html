À propos du projet

SAE 14

Ce site web a été conçu dans le cadre de ma formation en Réseaux et Télécommunications. L'objectif pédagogique est de maîtriser les fondamentaux du développement web : structuration HTML5, mise en forme CSS3.

Jean BROGGI

Passionné par l'informatique et l'histoire automobile. Ce projet me permet de lier la rigueur technique du code à l'esthétique du design italien.

Technologies utilisées :

  • HTML5 : Structure sémantique et accessible.
  • CSS3 : Flexbox pour la mise en page, animations fluides.
  • Git : Gestion de versions pour le suivi du projet.
