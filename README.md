## Description

Ce projet est un site web statique réalisé dans le cadre de la **SAE14**.
Il a pour thème la marque automobile **Alfa Romeo**, mettant en avant son histoire, ses modèles emblématiques et son esprit sportif.


## Fonctionnalités

* **Navigation :** Menu déroulant responsive.
* **Design :** Utilisation de la charte graphique Alfa Romeo (Rouge, Noir, Blanc).
* **Mise en page :** Utilisation de Flexbox pour le positionnement des éléments.
* **Animations CSS/JS :** Effets au survol et animations.
* **Pages intégrées :**
    * `index.html` : Page d'accueil.
    * `pages/modeles.html` : Présentation de la gamme.
    * `pages/histoire.html` : Historique de la marque.
    * `pages/scuderia.html` : Section course.
    * `pages/apropos.html` : Informations sur moi projet et le projet.

## Technologies utilisées

* **HTML5** : Structure sémantique (Header, Main, Footer, Nav).
* **CSS3** : Feuilles de styles externes, gestion des `z-index`, `position: fixed`, et media queries.
* **JavaScript** : Scripts d'animation légers (`anim.js`).
